02. Past Projects

ClassHill Fullstack

ClassHill was an all-in-one workplace for students to connect with peers and have organized discussions. The Slack-like platform enabled university students to communicate with people from their classes, student groups, and university campuses.

Tech Stack:
  • PHP back-end using Codeigniter framework and MVC architecture
  • NodeJS to enable real-time exchange of messages between users
  • MySQL to store user data including (e.g. posts, attachment uploads, voting, etc.)
  • Expo.dev to develop Android and iOS applications
  • Deployed on AWS Linux EC2 with VPC access controls for SSH and internal testing
  • Other AWS services included RDS for Database, S3 for storage, SES for programmatic SMTP emails, and Load Balancers for traffic handling
  • Google Analytics & Grafana to keep track of user behaviour and interactions

Newscoop Android

Newscoop provided an AI-powered summary of the biggest news from the last 48 hours. Users could simply tap or swipe through the app to keep reading. Newscoop's goal was to make news short and sweet and to provide an introduction to AI to filter large articles. Tech Stack:

  • PHP back-end using Codeigniter framework and MVC architecture
  • NewsApi.org was utilized to fetch for latest news articles at regular intervals via Cron Jobs
  • Aylien News AI used to compose news summaries and determine article tone, sentiment, and significance.
  • Android Application was programmed natively using Java (Android Studio)
  • MySQL database to store and analyze all news data

Forty Nine Twos Game Dev

A puzzle game for Android to familiarize myself with game development and design. Read my LinkedIn article on how I designed the game in MS PowerPoint and what were some of the best parts of this project.

Tech Stack:
  • GameMaker, a JavaScript based framework to design game physics, motions, levels and so on
  • MS PowerPoint to design all game assets - Yes, you read that right... MS PowerPoint!

RedPine Web

RedPine was a platform that allowed small musicians to find venues throughout Toronto and crowdfund concerts with their fanbase. At its peak, we were processing 10 shows a month at more than 50 different venue partners.

Tech Stack: React, JavaScript, MySQL, Python, Django

AShopApp Android

AShopApp let users scan the barcode of a retail product from H&M, Zara, and Forever21 to discover similar products from other stores near them.

Tech Stack: Android (Java), PHP (Codeigniter), JavaScript, MySQL, Cron Jobs
